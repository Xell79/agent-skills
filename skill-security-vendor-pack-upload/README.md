# Skill Security Vendor Pack

Commercial skill security review bundle for marketplace sellers, buyers, and agencies.

## Included

- `SKILL.md` skill instructions and workflow
- `scripts/skill_security_vendor_pack.py` CLI helper
- `references/output-contract.md` report schema reference
- `LICENSE.md` commercial license terms

## Runtime

- Run from the package root.
- Use `python` or `python3`.
- No third-party Python packages are required.

## Quick Start

```bash
python scripts/skill_security_vendor_pack.py \
  --skill-path "./some-skill-folder" \
  --out-json output.json \
  --out-md report.md
```
